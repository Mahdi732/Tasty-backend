# Backend Production Env Checklist (Ready To Fill)

Use this file as a variable-by-variable production checklist for Render.

How to use:

1. Open each service on Render.
2. Copy the service block from this file.
3. Fill values marked FILL_ME or OPTIONAL.
4. Paste variables in the Render Environment tab.
5. Check each checkbox after saving.

Notes:

- Values below are backend-repo scoped.
- Internal hostnames match backend/render.yaml.
- Do not commit real secrets.
